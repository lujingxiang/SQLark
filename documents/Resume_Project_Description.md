项目经历：开发 SQLark 量化分析智能体平台，构建自然语言→意图路由→工具调度→SQL 生成/执行/校验→量化分析受控流水线。基于 LangGraph StateGraph 与 FastAPI 实现、9 个tools工具（数据查询/聚合/Python 分析/形态/量价/波动率等），内置 K 线形态、量价关系、波动率三类分析模块及 MACD/RSI/布林带指标；采用 SSE 流式返回与对话历史持久化，提升交互体验与分析能力。
项目成果：
（1）支持 6 类语义意图（查询、分析、K 线形态、量价、波动率、闲聊）与 10+ 业务场景，SQL 生成成功率约 95%，SQL 校验层自动拦截非 SELECT 等风险语句；
（2）前端 SSE 流式首屏响应≤200ms，支持主题切换、对话历史切换与流式渲染平滑一致；
（3）系统可视化执行链路覆盖全部 16 个关键分析节点，支持 SQL 验证失败自动重试闭环，整体流程准确率显著提升；
（4）平台具备实时流式输出、会话历史管理与对话记录可追溯能力，支持 LangGraph 状态持久化
