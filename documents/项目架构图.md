## 六、数据流简图

```
用户输入 "宁德时代最近K线有金叉吗"
        │
        ▼
  [intent]    LLM 识别 → intent=pattern
        │
        ▼
  [planner]   LLM → {intent:"pattern", field:"LatestPrice", analysis_type:"pattern", limit:60}
        │
        ▼
[tool_selector]  LLM → 选择 PatternAnalysis 工具
        │
        ▼
[tool_executor]  执行 PatternAnalysisTool:
                 ① 查 OHLCV 数据 (SQL→SQLite)
                 ② 计算均线交叉 (MA5/MA20)
                 ③ 识别金叉/死叉信号
                 ④ 输出形态列表 + 交叉信号
        │
        ▼
[observation]   结果写入 state.pattern_result
        │
        ▼
  [decision]     硬编码: analysis_ok → summarize
        │
        ▼
  [summarize]    LLM → "宁德时代近期出现1次金叉（5日均线上穿20日），
                 伴随...(数据)...建议关注后续量能配合情况"
        │
        ▼
    SSE → 前端渲染（文字 + 图表）
```
